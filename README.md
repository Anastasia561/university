# University Management System

**University Management System** is a full-stack web application for managing students, courses, and academic resources. It features a secure backend built with Java Spring Boot and a responsive React frontend.

---

## Features

### User Roles
- **Student:** View enrolled courses, access course descriptions, track grades
- **Admin:** Manage users, courses, and academic resources

### Multilingual UI
- Supports **English**, **Polish**, and **Spanish**  
- Language can be dynamically switched from the user profile  

---

## Technologies Used

- **Frontend:** React, HTML5, CSS3, JavaScript  
- **Backend:** Java 21, Spring Boot, Spring Security, JWT
- **Database:** PostgreSQL  
- **Build Tool:** Maven 
- **Testing:** JUnit, Mockito, Testcontainers
- **Containerization:** Docker

---

## Performance & Security

- Passwords securely hashed  
- Role-based access control for all endpoints  
- 95% of page transitions complete in under 2 seconds
